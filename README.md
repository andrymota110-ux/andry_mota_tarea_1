# Calculadora de ISR en C++

## Descripción

Este proyecto consiste en un programa desarrollado en C++ que calcula el Impuesto Sobre la Renta (ISR) de un empleado según la escala vigente de la DGII en la República Dominicana.

## Funcionalidad

El programa permite:
- Ingresar el sueldo mensual del empleado.
- Calcular el sueldo anual.
- Determinar si aplica el ISR.
- Mostrar el monto del impuesto y el sueldo neto.
